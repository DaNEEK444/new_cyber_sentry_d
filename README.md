# Cyber Sentry Defense — Landing Page

This repository contains the official landing page for **Cyber Sentry Defense**, a cybersecurity awareness and protection service founded by Danika. The site is designed to clearly present services, pricing, and booking options for individuals and small businesses seeking practical, real‑world digital safety.

## Features
- Clean, modern landing page layout
- Service cards with clear descriptions and pricing
- Calendly integration for scheduling
- Ko‑fi integration for payments
- Tawk.to live chat widget
- Mobile‑responsive design
- Easy to customize and extend
- Microsoft Foundry - Cyber Sentry AI Agent

## Deployment
This site is deployed using **GitHub Pages**.

To publish:
1. Go to **Settings → Pages**
2. Set **Branch: main**
3. Save  
Your site will be live at:  
`https://<your-username>.github.io/<repository-name>/`
